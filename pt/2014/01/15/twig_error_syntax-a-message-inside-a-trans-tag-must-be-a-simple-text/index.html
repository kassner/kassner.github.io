Twig_Error_Syntax: A message inside a trans tag must be a simple text

2014-01-15 Português

Dica rápida:

Quando você tiver um código deste tipo:

{% trans %}prefix.{{ varname }}{% endtrans %}

E receber esta mensagem de erro:

Twig_Error_Syntax: A message inside a trans tag must be a simple text

Você pode usar o código abaixo como gambiarra alternativa.

{{ ("prefix." ~ varname)|trans }}

PS: é possível que esta não seja a melhor maneira de resolver.
